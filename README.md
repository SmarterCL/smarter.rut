# SmarterBOT

Plataforma de automatización y ERP basada en Odoo, n8n, Chatwoot, Supabase y Ollama. Chile · proyectos remotos y presenciales.

## Descripción

Este proyecto es el front-end de SmarterBOT, una plataforma integral de automatización y ERP. El sistema está construido sobre tecnologías como Odoo Enterprise, n8n para automatización, WhatsApp Cloud API, Chatwoot para atención al cliente, Ollama para IA local y Supabase como base de datos.

## Características

- Integración con Odoo Enterprise
- Automatización con n8n
- Integración oficial de WhatsApp
- IA local con Ollama
- Centro de agentes MCP
- Infraestructura propia o VPS gestionado
- Multi-empresa y multi-RUT desde el diseño
- Migración de canales de soporte a Chatwoot incluida

## Stack completo

- Odoo Enterprise
- n8n Automatización
- WhatsApp Cloud API
- Chatwoot
- Ollama IA
- Supabase
- VPS gestionado

## Planes disponibles

- **Startup**: Para emprendimientos
- **Pro**: Para operación completa  
- **Enterprise**: Para operación avanzada

## Instalación

El proyecto está configurado con pnpm como gestor de paquetes. Para ejecutarlo:

```bash
pnpm install
pnpm run dev
```
